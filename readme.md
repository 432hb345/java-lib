```xml

        <groupId>com.abranhe.bigdecimal</groupId>
```

## Usage

```java
import com.abranhe.allalgorithms.searches.*;

public static void main(String[] args) {
    int[] arr = {7, 2, 1, 19};
    
    MergeSort.sort(arr);
    // {1, 2, 7, 19}
}
```
